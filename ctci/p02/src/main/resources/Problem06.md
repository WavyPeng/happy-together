### [回文链表](https://leetcode-cn.com/problems/palindrome-linked-list-lcci/)

编写一个函数，检查输入的链表是否是回文的。

#### 示例1：
```
输入： 1->2
输出： false 
```

#### 示例2：
```
输入： 1->2->2->1
输出： true
```
