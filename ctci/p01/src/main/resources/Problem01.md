### [判定字符是否唯一](https://leetcode-cn.com/problems/is-unique-lcci/)

实现一个算法，确定一个字符串`s`的所有字符是否全都不同。

#### 示例1：
```
输入: s = "leetcode"
输出: false 
```

#### 示例2：
```
输入: s = "abc"
输出: true
```

#### 限制:
```
0 <= len(s) <= 100
如果你不使用额外的数据结构，会很加分。
```
